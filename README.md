## 🛠️ Common Windows Issues

If BrainBench does not open, try these steps:

- Right-click the app and choose Run as administrator
- Make sure the files are fully extracted
- Check that the dataset folder is still in the same place
- Re-download the files if the folder looks incomplete
- Restart your PC and try again

If Windows says the file is blocked, right-click the file, open Properties, and check whether there is an Unblock option near the bottom. If there is, select it and apply the change.

If the app opens but does not show data, confirm that you selected the correct dataset folder. Some tools need the full folder, not a single file.
